\chapter{Status}

Status beschreiben besondere Zustände, die eine Figur vorübergehend beeinflussen.

Ein Status gilt so lange, bis die bei ihm beschriebene Bedingung erfüllt ist oder ein anderer Effekt ihn beendet.

\section{Am Boden}

Eine Figur, die am Boden liegt, kann sich nur eingeschränkt verteidigen und bewegen.

\begin{itemize}
  \item Angriffe gegen eine am Boden liegende Figur haben Vorteil.
  \item Eine am Boden liegende Figur muss ihre nächste \moveAction{} verwenden, um aufzustehen.
  \item Sobald die Figur aufgestanden ist, endet der Status \emph{Am Boden}.
\end{itemize}

Weitere Status können später ergänzt werden.
