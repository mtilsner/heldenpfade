# Heldenpfade – Regelwerk

Dieses Verzeichnis enthält das Regelwerk von **Heldenpfade** als LaTeX-Projekt.

## Aufbau

- `main.tex` – Einstiegspunkt für das vollständige Regelwerk
- `heldenpfade.sty` – zentrale Formatdefinitionen und Makros
- `kapitel/` – Regelkapitel
- `anhang/` – Anhänge und Referenzmaterial
- `bilder/` – Abbildungen
- `icons/` – Symbole und grafische Spielelemente

## Kompilieren

Das Regelwerk wird über `main.tex` erzeugt.
